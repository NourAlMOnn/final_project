منصة فلاتر الوجه الاحترافية AR Face Filters
هذا المشروع هو تطبيق ويب متكامل يتيح للمستخدمين إضافة فلاتر الواقع المعزز AR على الوجه في الوقت الفعلي باستخدام الكاميرا أو من خلال رفع الصور

المميزات
ستة أقسام للفلاتر إكسسوارات مكياج تأثيرات خلفيات حيوانات وإبداع التحكم الذكي إمكانية التحكم في الحجم والشدة للفلتر الأخير المضاف فقط دعم الكاميرا والصور العمل مباشرة مع الكاميرا أو معالجة الصور المرفوعة حفظ النتائج إمكانية حفظ الصورة النهائية بجودة عالية بدون إنترنت كافة المكتبات محملة محلياً في مجلد lib

التقنيات المستخدمة
HTML5 CSS3 لبناء الواجهة التفاعلية JavaScript ES6 لمنطق التطبيق MediaPipe Face Mesh للكشف الدقيق عن ملامح الوجه ونقاطه 468 نقطة TensorFlow js لتشغيل نماذج الذكاء الاصطناعي داخل المتصفح

كيفية التشغيل
1 قم بفك ضغط الملف ar face project zip 2 افتح ملف index html في أي متصفح حديث يفضل Chrome أو Edge 3 اسمح بالوصول إلى الكاميرا عند الطلب 4 اختر الفلاتر من القائمة الجانبية واستمتع بالتجربة

ملاحظات تقنية
تم تنظيم الكود بشكل يمنع تداخل الفلاتر الصور المستخدمة للفلاتر تم إنشاؤها خصيصاً للمشروع بخلفيات شفافة المشروع يعمل بالكامل داخل المتصفح ولا يحتاج إلى خادم Server خاص ولكن يفضل تشغيله عبر Live Server لتجنب مشاكل CORS مع بعض المتصفحات
